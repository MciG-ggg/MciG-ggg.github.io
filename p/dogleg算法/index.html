1. 信赖域子问题

Dogleg 方法用于求解信赖域框架下的子问题：

$$ \min_{\mathbf{p} \in \mathbb{R}^n} \quad m(\mathbf{p}) = f + \mathbf{g}^T \mathbf{p} + \frac{1}{2} \mathbf{p}^T \mathbf{B} \mathbf{p}, \quad \text{s.t.} \quad \|\mathbf{p}\| \leq \Delta $$

其中：

  • $f = f(\mathbf{x}_k)$ 是目标函数在当前点 $\mathbf{x}_k$ 的值
  • $\mathbf{g} = \nabla f(\mathbf{x}_k)$ 是梯度向量
  • $\mathbf{B} = \nabla^2 f(\mathbf{x}_k)$ 是 Hessian 矩阵（或其近似）
  • $\Delta$ 是信赖域半径…

2. 关键方向

定义两个关键方向：

  1. 最速下降方向（Steepest Descent）： $$ \mathbf{p}_U = -\frac{\|\mathbf{g}\|^2}{\mathbf{g}^T \mathbf{B} \mathbf{g}} \mathbf{g} $$
  2. 牛顿方向（Newton Step）： $$ \mathbf{p}_B = -\mathbf{B}^{-1} \mathbf{g} $$

3. Dogleg 路径

Dogleg 方法通过构造一条分段路径连接 $\mathbf{p}_U$ 和 $\mathbf{p}_B$ ：

$$ \mathbf{p}(\tau) = \begin{cases} \tau \mathbf{p}_U, & 0 \leq \tau \leq 1 \\ \mathbf{p}_U + (\tau - 1)(\mathbf{p}_B - \mathbf{p}_U), & 1 \leq \tau \leq 2 \end{cases} $$

4. 最优步长选择

根据信赖域半径 $\Delta$ 的大小，选择不同的解：

  1. 当 $|\mathbf{p}_B| \leq \Delta$ 时： $$ \mathbf{p}^* = \mathbf{p}_B \quad (\text{完全牛顿步}) $$
  2. 当 $|\mathbf{p}_U| \geq \Delta$ 时： $$ \mathbf{p}^* = \frac{\Delta}{\|\mathbf{p}_U\|} \mathbf{p}_U \quad (\text{沿最速下降方向截断}) $$
  3. 其他情况：求解 $\tau$ 使得 $|\mathbf{p}(\tau)| = \Delta$ ，解为： $$ \tau = \frac{\mathbf{b} - \sqrt{\mathbf{b}^2 - \mathbf{a}(\mathbf{c} - \Delta^2)}}{\mathbf{a}} $$ 其中： $$ \mathbf{a} = \|\mathbf{p}_B - \mathbf{p}_U\|^2, \quad \mathbf{b} = \mathbf{p}_U^T (\mathbf{p}_B - \mathbf{p}_U), \quad \mathbf{c} = \|\mathbf{p}_U\|^2 - \Delta^2 $$

5. 几何解释

Dogleg 路径形成一个折线段：

  1. 从原点出发沿最速下降方向到 $\mathbf{p}_U$
  2. 从 $\mathbf{p}_U$ 沿直线到 $\mathbf{p}_B$

通过调整 $\tau$ ，在保证 $|\mathbf{p}| \leq \Delta$ 的前提下，找到使二次模型 $m(\mathbf{p})$ 最小的解。

6. 算法优势

  • 当 $\mathbf{B}$ 正定时，牛顿方向提供二阶收敛速度
  • 当 $\mathbf{B}$ 非正定时，通过最速下降方向保证稳定性
  • 计算复杂度低于精确求解信赖域子问题

Dogleg 算法：通俗解释与核心思想

1. 为什么要用 Dogleg 算法？

在优化问题中，我们常需要在“快速收敛”和“稳定性”之间做权衡：

  • 梯度下降法：稳定但收敛慢（像蜗牛爬山）
  • 牛顿法：收敛快但可能不稳定（像火箭发射，但容易偏离轨道）

Dogleg 算法结合了两者的优点，像一条灵活的“狗腿”路径，在保证稳定的同时，尽可能加速收敛。

2. 核心思想：两条路径 + 一个折线

Dogleg 的路径由两段组成：

  1. 最速下降段（保守路线）：沿梯度方向快速下降，确保稳定。
  2. 牛顿段（激进路线）：直接跳向牛顿方向的最优解，追求速度。

通过调整步长参数 $\tau$ ，在两条路径之间“折线切换”，最终在信赖域（安全区域）内找到最优解。

3. 通俗比喻：狗狗的折线奔跑

想象一只狗从家（初始点）出发找骨头（最优解）：

  • 第一段：先沿着最明显的路（梯度方向）跑，确保不迷路。
  • 第二段：发现远处有捷径（牛顿方向），直接冲刺，但不能超出绳子（信赖域）的长度。
  • 结果：在绳子范围内，选择最接近骨头的点。

4. 数学实现：分三步走

问题：在信赖域半径 $\Delta$ 内，求解二次模型的最小值：

$$ \min_{\|\mathbf{p}\| \leq \Delta} m(\mathbf{p}) = f + \mathbf{g}^T \mathbf{p} + \frac{1}{2} \mathbf{p}^T \mathbf{B} \mathbf{p} $$

关键步骤：

  1. 计算两个方向：

    • 最速下降方向 $\mathbf{p}_U$ ：沿梯度反方向，步长由梯度范数决定。
    • 牛顿方向 $\mathbf{p}_B$ ：直接求解 $\mathbf{B}\mathbf{p} = -\mathbf{g}$ 。

  2. 构造 Dogleg 路径：

    • 当 $\tau \in [0,1]$ ：沿最速下降方向走 $\tau \mathbf{p}_U$ 。
    • 当 $\tau \in [1,2]$ ：从 $\mathbf{p}_U$ 出发，向 $\mathbf{p}_B$ 直线移动。

  3. 选择最优解：

    • 情况 1：若牛顿方向 $\mathbf{p}_B$ 在信赖域内（ $|\mathbf{p}_B| \leq \Delta$ ），直接跳到 $\mathbf{p}_B$ 。
    • 情况 2：若最速下降方向超出信赖域（ $|\mathbf{p}_U| \geq \Delta$ ），沿梯度方向截断。
    • 情况 3：其他情况，通过解二次方程找到 $\tau$ 使得路径终点恰好落在信赖域边界。

  • 路径：从原点出发，先沿 $\mathbf{p}_U$ 到某个点，再转向 $\mathbf{p}_B$ 。
  • 信赖域：一个以原点为中心、半径为 $\Delta$ 的圆，解必须落在圆内。

6. 优势与局限性

优势：

  • 稳定高效：结合梯度下降的保守性和牛顿法的激进性。
  • 自动调参：无需手动设置学习率，通过信赖域半径控制步长。
  • 适合非线性最小二乘：常用于 SLAM、Bundle Adjustment 等场景。

局限性：

  • 依赖 Hessian 正定性：若 Hessian 矩阵 $\mathbf{B}$ 不正定，牛顿方向可能不存在。
  • 计算成本：需解线性方程组（但比高斯-牛顿法更高效）。

7. 实际应用示例

场景：机器人定位（非线性最小二乘问题）

  • 目标：通过传感器数据优化机器人的位置。
  • Dogleg 的作用：
    1. 初始阶段（远离最优解）：沿梯度方向保守下降，避免发散。
    2. 接近最优解时：切换牛顿方向，快速收敛到高精度解。

总结

Dogleg 算法像一位聪明的向导：

  • 在未知领域（远离解时），选择稳妥的梯度路径。
  • 在熟悉区域（接近解时），大胆使用牛顿方向加速。
  • 最终目标：在“安全范围”（信赖域）内，找到最优解的捷径。

import numpy as np

def dogleg_optimizer(func, grad, hess, x0, max_iter=100, tol=1e-6, delta_max=10.0):
    """
    Dogleg算法实现信赖域优化
    
    参数:
    func: 目标函数，接受numpy数组返回标量
    grad: 梯度函数，返回numpy数组
    hess: Hessian矩阵函数，返回numpy二维数组
    x0: 初始点（numpy数组）
    max_iter: 最大迭代次数
    tol: 收敛容差（梯度范数（范数表示该向量的长度）小于tol时停止）
    delta_max: 最大信赖域半径
    
    返回:
    x_opt: 最优解
    f_opt: 最优值
    iter_history: 迭代历史记录
    """
    x = x0.copy()
    delta = delta_max  # 初始信赖域半径
    iter_history = []
    
    for k in range(max_iter):
        # 计算当前点的函数值、梯度、Hessian
        f = func(x)
        g = grad(x)
        B = hess(x)
        
        # 检查收敛条件（梯度接近零）
        if np.linalg.norm(g) < tol:
            print(f"Converged after {k} iterations.")
            break
        
        # 计算最速下降方向 p_u
        p_u = -(np.dot(g, g) / np.dot(g, B @ g)) * g
        
        # 计算牛顿方向 p_b（需解线性方程组 B p = -g）
        try:
            p_b = np.linalg.solve(B, -g)
        except np.linalg.LinAlgError:
            # 若Hessian不可逆，退化为梯度方向
            p_b = p_u
        
        # 计算Dogleg路径上的最优步长
        if np.linalg.norm(p_b) <= delta:
            # 情况1：牛顿方向在信赖域内，直接使用
            p = p_b
        else:
            # 计算沿Dogleg路径的交点
            norm_pu = np.linalg.norm(p_u)
            if norm_pu >= delta:
                # 情况2：沿最速下降方向截断
                p = (delta / norm_pu) * p_u
            else:
                # 情况3：求解tau使得 ||p_u + (tau-1)(p_b - p_u)|| = delta
                # 二次方程 a*tau^2 + b*tau + c = 0
                a = np.dot(p_b - p_u, p_b - p_u)
                b = 2 * np.dot(p_u, p_b - p_u)
                c = np.dot(p_u, p_u) - delta**2
                tau = (-b + np.sqrt(b**2 - 4*a*c)) / (2*a)
                p = p_u + (tau - 1)*(p_b - p_u)
        
        # 计算实际下降量与预测下降量的比值 rho
        actual_reduction = func(x) - func(x + p)
        predicted_reduction = -(np.dot(g, p) + 0.5 * np.dot(p, B @ p))
        
        # 避免除以零（当预测下降量为零时跳过更新）
        if abs(predicted_reduction) < 1e-10:
            rho = 0.0
        else:
            rho = actual_reduction / predicted_reduction
        
        # 更新信赖域半径
        if rho < 0.25:
            delta = 0.25 * delta
        elif rho > 0.75 and np.linalg.norm(p) >= 0.95 * delta:
            delta = min(2*delta, delta_max)
        
        # 更新迭代点
        if rho > 0.01:
            x = x + p
        
        # 记录迭代历史
        iter_history.append((x.copy(), f, np.linalg.norm(g)))
    
    return x, f, iter_history

# 示例：使用Rosenbrock函数测试
def rosenbrock(x):
    return 100*(x[1]-x[0]**2)**2 + (1-x[0])**2

def rosenbrock_grad(x):
    return np.array([
        -400*x[0]*(x[1]-x[0]**2) - 2*(1-x[0]),
        200*(x[1]-x[0]**2)
    ])

def rosenbrock_hess(x):
    return np.array([
        [1200*x[0]**2 - 400*x[1] + 2, -400*x[0]],
        [-400*x[0], 200]
    ])

# 运行优化
x0 = np.array([-1.2, 1.0])
x_opt, f_opt, history = dogleg_optimizer(
    rosenbrock,
    rosenbrock_grad,
    rosenbrock_hess,
    x0,
    max_iter=50
)

print("最优解:", x_opt)
print("最优值:", f_opt)

代码使用说明

  1. 输入函数要求：

    • func ：标量函数（如最小二乘目标函数）
    • grad ：返回梯度向量
    • hess ：返回 Hessian 矩阵

  2. 适用场景：

    • 非线性最小二乘问题
    • Hessian 矩阵正定的问题（否则需加入阻尼项）

  3. 注意事项：

    • Hessian 矩阵可能不可逆时，需添加正则化（如 Levenberg-Marquardt）
    • 初始信赖域半径 delta_max 需根据问题调整
